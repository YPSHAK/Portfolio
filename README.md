## **Проекты и опыт**

## **Собственный продукт: платформа мульти‑агентов ИИ (в разработке, текущий основной проект)**

- Разрабатываю крупный веб‑инструмент — «арену» для взаимодействия различных ИИ‑агентов с гибкой архитектурой цепочек и продвинутой аналитикой.
- Реализованы базовые агентские функции с RAG, контекстом и thinking‑подходами, а также система многоуровневых агентов, которых можно собирать в цепочки под конкретные типы задач.
- Есть визуальный конструктор цепочек агентов прямо в веб‑интерфейсе, позволяющий комбинировать ИИ и скрипты в сложные сценарии.
- Реализована глубокая аналитика использования ИИ: лог запросов, стоимость и затраты, ошибки, метрики эффективности работы агентов.
- Разработаны режимы взаимодействия ИИ‑агентов между собой (дебаты, совместная работа, ответы по очереди и другие сценарии).
- Прорабатывается модуль симуляции событий (политические, экономические и др.): несколько ИИ в разных ролях разыгрывают сценарии, после чего агрегаторы и валидаторы строят общую картину возможного исхода.
- Использую OpenRouter API для удобного подключения разных моделей ИИ и экспериментов с архитектурой агентов.
